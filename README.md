# DocTools
### AI-powered documentation and developer tools
Built by Łukasz Pekala — 15 years in technical documentation, now building the tools that do it.

---

## Why I'm building this

I spent 15 years writing, structuring, and maintaining technical documentation
for engineering products used across Europe. I know the problems intimately:

- Documentation that drifts out of sync with the product
- Style inconsistencies no one has time to fix manually
- Knowledge locked in formats no one can search
- Review cycles that take days and still miss things

AI changes what's possible. DocTools is my answer to these problems —
a portfolio of tools built to solve real documentation pain points,
using modern web technology and AI APIs.
